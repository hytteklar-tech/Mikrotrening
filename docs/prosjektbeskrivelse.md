# Prosjektbeskrivelse — Sommerkropp

## Problem

Folk vil trene, men fullverdige treningsøkter føles overveldende. Uten et enkelt system for å bygge vane og se fremgang faller man fort ut. Spesielt travle hverdager ender uten aktivitet.

## Verdiforslag

Sommerkropp gjør mikrotrening til en daglig vane gjennom:
- **Enkelhet** — Én knapp for å registrere today's workout
- **Motivasjon** — Streak, badges og leaderboard blant venner
- **Lavterskel** — Treningspakker på 50 reps styrkeøvelse er nok
- **Sosial** — Se om venner er aktive i dag

## Målgruppe

Venner og familie som ønsker å komme i gang med regelmessig aktivitet, uten å forplikte seg til tunge treningsprogram. Delt privat via invitasjonskode — ikke en offentlig plattform.

## Teknisk scope

- Mobilapp / PWA
- Innlogging med mobilnummer (OTP/SMS)
- Datalagring per bruker i sky
- Grupper via invitasjonskode
