# MVP-Scope — Sommerkropp v1

## Inn i MVP (v1)

| Modul | Begrunnelse |
|-------|-------------|
| M-001 Treningspakker | Kjernefeature — uten dette finnes ikke appen |
| M-002 Daglig registrering | Kjernefeature |
| M-003 Streak + badges | Viktigste motivasjonsverktøy |
| M-004 Innlogging SMS/OTP | Nødvendig for datalagring per bruker |
| M-005 Datalagring | Nødvendig |
| M-006 Sosial (aktiv/ikke aktiv) | Lav kompleksitet, høy motivasjonsverdi |

## Leaderboard i MVP?
Leaderboard (del av M-003) tas med i MVP — det er enkelt å implementere med poeng-beregning on-the-fly og er en kjernedel av motivasjonen.

## Utenfor MVP (v2)

| Modul | Hvorfor utsatt |
|-------|---------------|
| M-007 Statistikk/historikk | Nyttig, men ikke kritisk for dag 1 |
| M-008 Push-påminnelser | Web Push krever mer oppsett, kan legges til i v1.1 |

## Out-of-scope (aldri i v1)
- Avanserte treningsprogram / periodisering
- Video-veiledning
- Betaling / premium
- Offentlig profil
